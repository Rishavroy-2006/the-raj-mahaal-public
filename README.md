# The Raj Mahaal - Royal Banquet Hall

Welcome to the frontend application for **The Raj Mahaal**, Sodepur's premier fully air-conditioned banquet hall for weddings, receptions, engagements, and corporate events.

## Features

- **Modern & Luxurious Design:** Crafted with high-contrast elegant typography and a luxury-gold color palette.
- **Responsive Layout:** fully optimized for desktops, tablets, and mobile devices.
- **Rules and Regulations Page:** Clear, dedicated section for the venue's rules, integrated via seamless routing.
- **Interactive Forms:** Fully styled contact form optimized for dark themes.
- **Smooth Animations:** Built with Framer Motion for elegant scroll effects and transitions.

## Tech Stack

- **React 19**
- **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **React Router v7** (Navigation)
- **Lucide React** (Icons)

## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Directory Structure

- `/src` - Application source code
  - `/components` - Reusable React components (Layouts, Sections, UI)
  - `/data` - Static data and configuration
  - `/pages` - Route components (Home, Rules and Regulations)
  - `/utils` - Utility functions
- `/public` - Static assets and images
- `/scripts` - Utilities for setting up project images

## Customization

You can update the branding, contact details, and core content inside `src/data/config.ts`.
